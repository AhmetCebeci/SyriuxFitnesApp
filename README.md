# Spor Salonu Yönetim Sistemi (FitnessApp)

**Ders:** Web Programlama
**Öğrenci:** Ahmet Cebeci
**Numara:** B231210077
**Grup:** C1 Grubu

## Proje Hakkında
Bu proje, Sakarya Üniversitesi Web Programlama dersi kapsamında **ASP.NET Core MVC 8.0** kullanılarak geliştirilmiş bir **Spor Salonu Yönetim ve Randevu Sistemi**dir.

Proje içerisinde; antrenör ve hizmet yönetimi, online randevu sistemi, REST API entegrasyonu ve yapay zeka destekli egzersiz önerileri bulunmaktadır.
Sistem, Admin ve Üye olmak üzere iki farklı rol tabanlı yetkilendirme ile çalışmaktadır.
